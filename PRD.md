# Lane Changer — PRD (제품 요구사항 정의서)

> 이 문서는 Lane Changer의 제품 목표, 기능 요구사항, 기술 결정 근거를 정의합니다.

---

## 1. 제품 개요

### 배경
리그 오브 레전드 유저들은 포지션 변경을 고민하지만, 막연한 감에 의존하는 경우가 많다.
기존 전적 사이트(op.gg, 포로지능 등)는 현재 성적을 보여줄 뿐, "다른 포지션을 해봐"라는 제안은 하지 않는다.

### 목표
- 전적 데이터를 분석해 유저에게 가장 잘 맞는 포지션을 추천한다.
- 단순 추천이 아닌, 냉혹하고 직설적인 팩폭 코멘트로 동기를 부여한다.
- 검색 한 번으로 3~10초 내에 결과를 제공한다.

### 성공 지표
- 검색 완료율 (에러 없이 결과까지 도달한 비율)
- 재방문율 (같은 유저가 다시 검색하는 비율)
- 공유율 (결과 화면을 캡처/링크 공유하는 비율)

## 3. 기능 요구사항

### 3-1. 핵심 기능 (MVP)

#### F-01. 소환사 검색
- `소환사명#태그` 형식 입력
- 태그 생략 시 `KR1` 자동 적용
- Enter 키 또는 버튼 클릭으로 분석 시작
- 최근 검색 5개 드롭다운 표시 (localStorage)

#### F-02. 전적 수집
- Riot Games API v5 사용
- 랭크 솔로 큐(420) 우선 조회, 5게임 미만이면 전 큐 포함
- 최대 20게임 수집
- 5개씩 배치 조회 (딜레이 1.2초, Rate Limit 방지)

#### F-03. 포지션 분석
- 5개 포지션(탑/정글/미드/원딜/서포터) 적합도 점수 계산
- 주포지션 판단: 전체 게임의 40% 이상 플레이한 포지션
- 점수 정규화: 0~100점 (고승률 케이스 최대 50점)

**분석 지표**

| 지표 | 활용 포지션 |
|------|------------|
| KDA | 전체 |
| CS/분 | 탑, 미드, 원딜 |
| 킬 관여율 | 정글, 서포터 |
| 딜량 | 미드, 원딜, 정글 |
| 피해 흡수량 | 탑, 서포터 |
| 시야 점수 | 서포터 |
| CC 시간 | 탑, 정글, 서포터 |
| 드래곤/바론 테이크다운 | 정글 |
| 오브젝트 스틸 | 정글 |
| 힐량 | 서포터 |

#### F-04. 결과 분기 처리

| 케이스 | 조건 | 처리 |
|--------|------|------|
| 일반 | 주포지션 있음 + 승률 55% 미만 | 4개 포지션 추천 (0~100점) |
| 고승률 | 주포지션 있음 + 승률 55% 이상 | 1개만 추천 (0~50점) |
| 멀티포지션 | 주포지션 없음 (40% 미만) | 1개만 추천, "당장 이것만 파" |

#### F-05. 플레이스타일 태그
10가지 태그 자동 분류: 높은 KDA / 팀 기여형 / 파밍 지향 / 딜 지향 / 탱커형 / CC 활용 / 시야 관리 / 힐·보호 / 정글 경험 / 밸런스형

#### F-06. 챔피언 추천
- 포지션별 챔피언 풀 8개 보유
- 유저 플레이스타일 태그 ↔ 챔피언 태그 매칭 점수로 3개 선정
- Data Dragon API로 챔피언 이미지 표시

#### F-07. AI 팩폭 코멘트
- Claude API (claude-sonnet-4-6) 사용
- 반말 + 사이코패스 코치 스타일
- 케이스별 다른 지시문 사용
- 과부하 시 1.5초 간격 최대 3회 재시도
- 3회 실패 시 코멘트 없이 결과만 표시

## 4. 기술 결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Next.js 16 (App Router) | 프론트+API 통합, Vercel 최적화 |
| 백엔드 | Next.js API Routes | FastAPI 별도 서버 불필요, 간소화 |
| 배포 | Vercel | GitHub 연동 자동 배포 |
| DB | Supabase | 추후 캐싱 용도 (현재 미사용) |
| 전적 API | Riot Games API v5 | 공식 API, 무료 |
| AI | Claude API (Sonnet 4.6) | 자연어 코멘트 생성 |
| 검색 기록 | localStorage | 서버 저장 불필요, 개인화 |
| 챔피언 이미지 | Data Dragon CDN | 공식 무료 이미지 |

## 5. 제약사항

| 제약 | 내용 |
|------|------|
| Riot API 키 | Development Key 24시간 만료 → Production Key 신청 필요 |
